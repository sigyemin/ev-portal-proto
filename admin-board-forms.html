DEV개발 참고 — 신설 게시판·분류 코드 (실서비스 비노출)
  • 신설 게시판 : 서식다운로드 — AS-IS BLBD(게시판 마스터)+ARTC(게시물) 공용 구조에 게시판 구분 신설. TNCM_MENU에도 신규 메뉴 등록 필요(시스템관리 > 게시판 관리, 질의응답 다음 순번).
  • 서식 분류 = 코드그룹 FORM_CATE(안) 8종, 하드코딩 금지(FAQ 전철) · 코드값·채번 미정(개발 협의) · 순서 SORT_ORDR
  • 게시상태 = ARTC.OPEN_YN — 노출/미노출. ★미노출은 대민에서만 제외되고 관리자 조회는 계속 유지(아무에게도 안 보이는 상태가 아님).
  • 비밀글(ARTC.SECRET)은 질의응답 전용 — 본 게시판 미적용. 게시상태(OPEN_YN)와 혼동 금지.
  • ★BLBD.OPEN_YN='Y' 필수 — 'N'이면 boardEdit 등록화면에 노출여부 라디오가 렌더링되지 않아 노출 제어 자체가 불가(boardEdit.jsp:307~311). 전 게시판 'Y'.
  • 공지글 미사용 — 공지사항 게시판 전용 기능(목록 상단 고정).
  • 데이터 원천 공유 : 대민 누리집 서식다운로드 페이지(정보 자료실)와 동일 게시판 데이터 사용 — 관리자 등록·수정이 대민에 그대로 노출.
  • AS-IS 공용 패턴 : boardList(구분·처리상태·작성일·검색어 필터) / boardEdit(게시판 구분·구분(카테고리)·제목·내용·게시상태·첨부파일) 재현 — 게시판 구분='서식다운로드' 고정 파라미터.
