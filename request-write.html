🛠 개발 참고 — 요청자료 원천·코드값

요청자료 원천 = EVCS_REQUEST_DATA (실컬럼 14종)

요청구분 = REQ_CATE (1충전기 · 2보조금 · 3기타)

처리상태 = REQ_STATUS (1신청 · 2승인 · 3반려 · 4처리중 · 5처리완료) — 등록 시 1 신청으로 생성

코드 = EVCS_CODE_DETAIL (계층은 UP_CODE, USE_YN 없음, 정렬 SORT_ORDR)

‘요청기관’ 입력칸 제거됨(2026-07-24 확정) — EVCS_REQUEST_DATA에 대응 컬럼 없음. 향후 필요 시 컬럼 신설 또는 회원 소속기관 대체가 전제.
